{% trans "Overview" %}

{% blocktrans %}Sometimes it is useful to communicate directly and without delay in order to achieve a common goal. Or it may be useful to direct interested people from the outside to an accessible and open communication room, where they can meet the people involved in a group or project.{% endblocktrans %}

{% blocktrans %}Chat or Instant Messaging can be a useful tool under these circumstances.{% endblocktrans %}

{% blocktrans %}A wide variety of instant messenger applications and protocols is available, e.g. Jabber, IRC, Whatsapp, Facebook Messenger, Slack, Mattermost and many more. Grouprise integrates the Matrix protocol, since it offers a wide range of features, that turn it into a perfect fit for this platform. These features are detailed below.{% endblocktrans %}

  • {% trans "grouprise" %}: {% trans "The community website your are browsing right now is based on the grouprise software." %}
  • {% trans "Matrix" %}: {% trans "Matrix is a decentral and federated chat communication protocol." %}

{% trans "Matrix Features" %}

{% trans "Matrix offers the following features:" %}

  • {% trans "trustworthy" %}: {% trans 'Both the server and the client implementations are available as Free Software. Thus anyone interested can verify, that the software is operating in the interest of the users.' %}
  • {% trans "federated" %}: {% trans "Everyone can host their own server and all of these servers communicate with each other. This is similar to the design of the email protocol or the telephone network, where users of different providers can communicate with each other." %}
  • {% trans "secure" %}: {% trans "End-to-end encryption keeps the content of conversations safe from pryring eyes. Even the service provider cannot peek inside." %}
  • {% trans "wide range of clients" %}: {% trans 'Pick your favorite client from a wide range of applications for various mobile and desktop platforms.' %}

{% trans "Integration with community website" %}

{% trans "The integration of the Matrix chat into a grouprise instance (this community website) offers the following features:" %}

  • {% trans "Use your grouprise user account (name and password) to log in to the Matrix homeserver. There is no need for a separate Matrix account." %}
  • {% blocktrans with matrix_client_url=matrix_client_path|full_url %}A local instance of the Element web client is provided along with the grouprise community website ({{ matrix_client_url }}).{% endblocktrans %}
  • {% trans "A public and a private Matrix room is automatically created for each grouprise group." %}
  • {% trans "All members of a grouprise group are automatically invited to the group's Matrix rooms." %}
  • {% trans "Notifications for new content (articles, events, discussions, ...) are sent to the related Matrix rooms (internal messages are announced in the private room, only)." %}

{% trans "Additionally all regular features of the Matrix communication ecosystem are available:" %}

  • {% trans "Matrix users from other servers may join your group's rooms (if you invite them)." %}
  • {% trans "You may use the Matrix account provided by grouprise or your existing (external) Matrix account." %}
  • {% trans "You are free to communicate with your group peers or with any other Matrix user in the world." %}

{% trans "FAQ (Frequently Asked Questions)" %}

{% trans "Why is there a public and a private room?" %}

{% blocktrans %}These pre-configured rooms for each group were prepared for your convenvience. The public room is configured to be open for random visitors. Such rooms are visible in the public "room directory" (list of rooms hosted on the Matrix server). A private room is not listed in the room directory of the server. Interested people may only join private rooms after being invited by a current room member.{% endblocktrans %}

{% trans "Room members" %}

{% blocktrans %}Whenever someone joins a group in the grouprise website, the corresponding matrix account of this new member automatically is invited to the public and the private Matrix room belonging to this group.{% endblocktrans %}

{% blocktrans %}People leaving a grouprise group are automatically removed from the group's Matrix rooms.{% endblocktrans %}

{% blocktrans %}Any external person (even people without an account on the local grouprise instance) may join public rooms or may be invited to private rooms.{% endblocktrans %}
