Repository and issue reporting
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Development of the software happens on in our gitlab repository:
  `git.hack-hro.de/grouprise/grouprise <https://git.hack-hro.de/grouprise/grouprise/>`_

Here you can report issues and propose contributions.

Discussions
^^^^^^^^^^^

Please visit our matrix chat rooms if you have a question or you want to discuss an idea:

`#grouprise-dev:systemausfall.org <matrix:r/grouprise-dev:systemausfall.org>`_
  development and plans
`#grouprise-ops:systemausfall.org <matrix:r/grouprise-ops:systemausfall.org>`_
  operating a grouprise instance
`#grouprise-users:systemausfall.org <matrix:r/grouprise-users:systemausfall.org>`_
  using grouprise and proposing ideas
